# Summarizer dev

Welcome to **Summarizer dev**, a versatile tool for summarizing text using various summarization models. This app allows you to upload text, enter a URL, or type directly into a text area and choose a summarization model to generate concise summaries of your content.

## Features

- **Multiple Summarization Models**: Choose from NLTK, Spacy, Gensim, and Sumy models.
- **File Upload**: Upload `.txt` and `.pdf` files for summarization.
- **URL Summarization**: Enter a URL to fetch and summarize content from web pages.
- **Text Area Input**: Manually input text to be summarized.
- **Download Summaries**: Download the generated summaries as text files.
- **Future Tasks**: Planned features and improvements include:
  - **Video Summarization**: Integrate video processing capabilities for summarizing audio-visual content.
  - **Custom Summarization Settings**: Allow users to adjust summarization parameters and length.

## How to Use

1. **Choose Task**: Select either "Summarize" or "Compare Summarizers" from the sidebar.
2. **Choose Model**: Select your preferred summarization model from the dropdown menu.
3. **Input Text**:
    - Upload a file (`.txt` or `.pdf`).
    - Enter a URL to fetch content.
    - Type or paste text into the text area.
4. **Generate Summary**: Click the "Summarize" or "Compare" button to generate the summary.
5. **Download Summary**: Click the download button next to each summary to save it as a text file.
